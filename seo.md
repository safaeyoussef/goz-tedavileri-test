# Search Engine Optimization Strategy: goztedavileri.com

## 1. Technical SEO Requirements
- **Language:** HTML lang="tr" (Turkish).
- **Architecture:** Semantic HTML5 tags exclusively (<article>, <section>, <nav>, <main>).
- **Crawlability:** Auto-generate `sitemap.xml` and `robots.txt` via Astro plugins.
- **Metadata:** Dynamic Title Tags and Meta Descriptions for all 21+ treatment pages.

## 2. Page Specific SEO
- **H1 Logic:** Every page must have exactly one H1 containing the primary keyword (e.g., "SMILE Göz Ameliyatı Hakkında Her Şey").
- **Image Optimization:** All images must be in `.webp` or `.avif` format with descriptive `alt` text.
- **Internal Linking:** Every treatment page must link back to the "Göz Testi" or "Güncel Haberler" to keep users in the ecosystem.

## 3. Definitions for Implementation
- **Hydration:** The process of adding interactivity to static HTML. We will use "Partial Hydration" to keep the site fast.
- **Semantic HTML:** Using tags that describe their content (e.g., <nav> for menus) so Google understands the site structure perfectly.
